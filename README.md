\# Loja Virtual com PrestaShop



Projeto desenvolvido para a disciplina de Desenvolvimento de E-commerce com CMS.



\## 📌 Descrição

Este projeto consiste na criação de uma loja virtual utilizando o PrestaShop em ambiente local com XAMPP.



\## ⚙️ Tecnologias utilizadas

\- XAMPP (Apache + MySQL)

\- PHP 7.4

\- PrestaShop 1.7.8.10



\## 🛠️ Funcionalidades

\- Cadastro de produtos

\- Organização por categorias

\- Carrinho de compras

\- Sistema de pagamento

\- Configuração de entrega



\## ⚠️ Observações

Durante o desenvolvimento foram encontrados erros de compatibilidade com versões mais recentes do PHP, sendo necessário utilizar versões mais estáveis.
